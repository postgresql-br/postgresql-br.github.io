Chamada de trabalhos para o PGCon Brasil 2009

Date 31/05/2009 10:20 By telles Tags PGCon Brasil 2009

Está aberta a chamada de trabalhos para o PGCon Brasil 2009. O período de inscrições vai de 30/05/2009 até 30/06/2009. Como novidade este ano, você pode escolher entre 4 modalidades: tutorial, hacker talk, palestra normal e palestra avançada. Só para constar, sim, haverá uma chamada de trabalhos para os Lightning Talks, mas a exemplo do ano passado, ela só será aberta uma semana antes do evento.

Se você desenvolve um trabalho interessante com PostgreSQL ou conhece alguém que o faça, não perca esta oportunidade de enriquecer um dos maiores eventos de PostgreSQL do mundo!
